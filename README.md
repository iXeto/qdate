# QDate

Fresh hackathon build of QDate, a dating app.

## Hackathon Status

This repository was initialized during the AI BEAVERS founder hackathon on June 6, 2026.
It is a fresh build and does not import an existing private product codebase.

Public repo: https://github.com/iXeto/qdate

## Current App State

- React Native app scaffolded with Expo Router.
- Initial home screen shows the QDate title.
- Placeholder button is present and intentionally has no product behavior yet.

## Stack

- Expo SDK 54
- React Native 0.81
- React 19
- Expo Router
- TypeScript

## Run

```sh
npm install
npm run start
```
